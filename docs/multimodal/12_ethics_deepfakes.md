# Ethics & Deepfakes

## The Danger
-   Non-consensual imagery.
-   Political disinformation (fake speeches).
-   Social Engineering (Voice cloning CEO).

## Solutions
1.  **Watermarking**:
    -   *Visible*: DALL-E watermark.
    -   *Invisible*: SynthID (modifying logits).
2.  **C2PA / Content Credentials**: Cryptographically signing media provenance (Camera -> Editor -> Web).
3.  **Hiding Concepts**: Machine Unlearning (Making the model forget "NSFW" concepts).
